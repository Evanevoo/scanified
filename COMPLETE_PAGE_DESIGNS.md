# Complete Page Designs - Gas Cylinder App

## Design System Foundation

### Color Palette
- **Primary Teal**: `#40B5AD` - Main brand color (buttons, links, accents)
- **Primary Dark**: `#2E9B94` - Hover states
- **Secondary Turquoise**: `#48C9B0` - Accent color
- **Light Teal**: `#5FCDC5` - Highlights and gradients
- **Success Green**: `#10B981` - Success states
- **Warning Amber**: `#F59E0B` - Warning states
- **Error Red**: `#EF4444` - Error states
- **Text Primary**: `#111827` (Gray 900)
- **Text Secondary**: `#6B7280` (Gray 600)
- **Text Muted**: `#9CA3AF` (Gray 400)
- **Borders**: `#E5E7EB` (Gray 200)
- **Backgrounds**: `#FFFFFF` / `#F9FAFB` (Gray 50)

### Typography Scale
- **H1**: `text-5xl md:text-6xl lg:text-7xl font-bold text-gray-900`
- **H2**: `text-4xl md:text-5xl font-bold text-gray-900`
- **H3**: `text-2xl md:text-3xl font-semibold text-gray-900`
- **H4**: `text-xl font-semibold text-gray-900`
- **Body Large**: `text-xl md:text-2xl text-gray-600`
- **Body**: `text-base text-gray-600`
- **Body Small**: `text-sm text-gray-600`
- **Caption**: `text-xs text-gray-500`

### Spacing Scale
- **Section Padding**: `py-24` (96px) for major sections
- **Card Padding**: `p-6` (24px) for card content
- **Gap**: `gap-6` (24px) for grids, `gap-4` (16px) for flex items
- **Margin Bottom**: `mb-8` (32px) for headings, `mb-4` (16px) for paragraphs

## Component Patterns

### Buttons
```jsx
// Primary
<Button className="bg-[#40B5AD] hover:bg-[#2E9B94] text-white rounded-xl px-6 py-3">

// Secondary
<Button className="border-2 border-gray-300 text-gray-700 hover:bg-gray-50 rounded-xl px-6 py-3">

// Ghost
<Button className="text-[#40B5AD] hover:text-[#2E9B94] hover:bg-[#40B5AD]/10 rounded-xl px-4 py-2">
```

### Cards
```jsx
<Card className="bg-white border border-gray-200 rounded-xl p-6 hover:border-[#40B5AD]/30 hover:shadow-lg transition-all">
```

### Inputs
```jsx
<input className="w-full border-2 border-gray-300 rounded-xl px-4 py-3 focus:border-[#40B5AD] focus:ring-2 focus:ring-[#40B5AD]/20 outline-none transition-all" />
```

### Badges
```jsx
// Status badges
<Badge className="bg-[#10B981]/10 text-[#10B981] rounded-full px-3 py-1 text-xs font-semibold">
// Active
<Badge className="bg-[#F59E0B]/10 text-[#F59E0B] rounded-full px-3 py-1 text-xs font-semibold">
// Warning
<Badge className="bg-[#EF4444]/10 text-[#EF4444] rounded-full px-3 py-1 text-xs font-semibold">
// Error
```

### Tables
```jsx
<table className="w-full border border-gray-200 rounded-xl overflow-hidden bg-white">
  <thead className="bg-gray-50 text-gray-900 font-semibold">
    <tr>
      <th className="px-6 py-4 text-left">Column 1</th>
      <th className="px-6 py-4 text-left">Column 2</th>
    </tr>
  </thead>
  <tbody>
    <tr className="hover:bg-gray-50 border-b border-gray-200">
      <td className="px-6 py-4">Data 1</td>
      <td className="px-6 py-4">Data 2</td>
    </tr>
  </tbody>
</table>
```

## Responsive Design

### Mobile (< 768px)
- Single column layouts
- Stacked cards
- Full-width buttons
- Simplified navigation (hamburger menu)
- Bottom navigation for key actions
- Collapsible sections

### Tablet (768px - 1024px)
- Two-column grids
- Side-by-side forms
- Compact tables with horizontal scroll
- Expanded sidebar

### Desktop (> 1024px)
- Three-column grids
- Full feature sets
- Expanded tables
- Persistent sidebar
- Multi-column layouts

## Animation Guidelines

### Page Transitions
- Fade in: `opacity: 0 â†’ 1` (0.5s)
- Slide up: `y: 20 â†’ 0` (0.5s)
- Stagger: `delay: index * 0.1` for lists

### Hover Effects
- Cards: `translateY(-4px)` + shadow increase
- Buttons: `scale(1.02)` + shadow increase
- Links: Color change + underline

### Loading States
- Spinner: Rotating circle with teal color
- Skeleton: Pulsing gray boxes
- Progress: Animated bar with gradient
